Crear los building blocks reutilizables ANTES de cualquier CRUD:

Componente Ubicación Propósito

FormModal.vue components/shared/ Modal genérico para crear/editar
ConfirmDialog.vue components/shared/ Confirmar eliminación
SearchInput.vue components/shared/ Input con debounce
ParamSelect.vue components/shared/ Select que carga opciones de tablas param
UbigeoSelect.vue components/shared/ Selects encadenados País > Depto > Prov > Dist
StatusBadge.vue components/shared/ Badge activo/inactivo
EmptyState.vue components/shared/ Estado vacío
DatePicker.vue components/shared/ Selector de fecha

Composable Ubicación Propósito

Tipo Ubicación

Checklist por Entidad (16 pasos)

  • 1. DTO Create → app/DTOs/{Dominio}/Create{Entidad}DTO.php
  • 2. DTO Update → app/DTOs/{Dominio}/Update{Entidad}DTO.php
  • 3. Service → app/Services/{Dominio}/{Entidad}Service.php
  • 4. FormRequest Store → app/Http/Requests/{Dominio}/Store{Entidad}Request.php
  • 5. FormRequest Update → app/Http/Requests/{Dominio}/Update{Entidad}Request.php
  • 6. Controller → app/Http/Controllers/{Dominio}/{Entidad}Controller.php
  • 7. Rutas → routes/web.php
  • 8. Types TS → resources/js/types/models/{entidad}.ts
  • 9. Component Form → resources/js/components/{dominio}/{Entidad}Form.vue
  • 10. Page Index → resources/js/pages/{dominio}/Index.vue
  • 11. Page Create → resources/js/pages/{dominio}/Create.vue
  • 12. Page Show → resources/js/pages/{dominio}/Show.vue (si aplica)
  • 13. Page Edit → resources/js/pages/{dominio}/Edit.vue
  • 14. Wayfinder → php artisan wayfinder:generate
  • 15. Test → tests/Feature/{Dominio}/{Entidad}ControllerTest.php
  • 16. Pint → vendor/bin/pint --dirty --format agent
PRECAUCIÓN:
NUNCA poner lógica de negocio en controllers.
NUNCA validar en controllers. Solo FormRequests.
NUNCA hardcodear URLs en el frontend. Usar Wayfinder.
